## How to Use

1. Click any day to mark it complete (green)
2. Click again to unmark
3. Build your streak by completing consecutive days
4. Your progress is saved automatically

## Tech Stack

- Next.js 16 + React 19
- TypeScript
- Tailwind CSS
- LocalStorage (no backend needed)

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Deploy to Vercel in one click:

```bash
vercel
```

## Future Features (Optional)

Want multi-device sync? Add Supabase:
1. Create Supabase project
2. Add env vars: `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Habits sync across all devices

For now, it's 100% local and free!
