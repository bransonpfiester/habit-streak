## How to Use

### First Time Setup
1. Click "+ Add Habit" to create your habits (e.g., Gym, Cook, Read)
2. Each habit gets a color

### Daily Use
**Year View:**
- Each day shows colored dots for your habits
- Click a dot to toggle that habit on/off
- See your entire year at a glance

**Weekly View:**
- Toggle to see current week in detail
- Checkboxes for each habit per day
- Navigate between weeks with Previous/Next
